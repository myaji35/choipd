# Kakao Pro 법무 체크리스트

**작성일**: 2026-04-21 · **상태**: 초안 (LEGAL_REVIEW 필요)

## 1. 카카오 비즈니스 채널 ToS

| 항목 | 검토 결과 | 조치 |
|---|---|---|
| 제3자 서버가 메시지 수신/저장 허용 | ⚠️ 카카오 i Connect 파트너 가입 필요 (대기업 위주) | 비즈니스 채널 API 직접 사용 — 메시지 발신만 가능. 수신은 별도 webhook 필요 |
| 자동 메시지 발송 | ❌ 상업적 자동발송 일부 제한 | imPD는 발송 안 함 (읽기/분석/알림만) |
| OAuth 권한 범위 | 채널 메시지 읽기 권한 (talk_message + channel_read) | 사장님이 직접 OAuth로 imPD에 권한 부여 |

## 2. 개인정보보호법 준수

| 항목 | 처리 방안 |
|---|---|
| **수집 대상** | 카카오 채널 대화 내용 (사장님-고객 양방향) |
| **수집 근거** | 사장님(개인정보처리자)의 imPD(수탁자) 위탁 |
| **수탁 계약** | imPD ↔ 사장님 간 개인정보처리위탁 계약서 체결 (Pro 가입 시 동의) |
| **고객 고지** | 사장님이 채널 프로필에 "AI 보조 사용 중" 표기 (필수) |
| **보관 기간** | **7일 후 원문 자동 파기** (요약/통계만 보존) |
| **PII 분리** | 전화번호/주소/주민번호는 별도 마스킹 처리 |
| **국외 이전** | LLM API(OpenAI/Anthropic) 사용 시 별도 동의 필요 → 한국어 LLM(HyperCLOVA X) 우선 검토 |
| **삭제권** | 고객이 사장님에게 삭제 요청 시 → 사장님이 imPD 대시보드에서 즉시 삭제 |

## 3. 정보통신망법

| 항목 | 검토 |
|---|---|
| **부가통신사업자 신고** | imPD가 "전자적 통신 내용 취급" → 미래창조과학부 신고 필요 가능 |
| **암호화** | 전송 중 TLS 1.3 + 저장 시 AES-256 |
| **접근 통제** | 사장님 본인만 접근 (Devise 인증 + 세션) |
