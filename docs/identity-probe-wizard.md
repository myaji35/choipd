# Identity Probe Wizard — 6스텝 위자드 UI/UX

**경로**: /auth/signup → submit → /welcome/probe → /<slug>
**총 소요**: 평균 2:30 (6스텝 × 25s)
**철학**: 회원은 관찰자·결정권자. 입력이 아니라 결정만.

## 5계명
1. "입력" 대신 "체크" — 칩/스위치/카드 탭 중심
2. 언제든 탈출 — 상단 우측 "건너뛰기" 상시 노출
3. 한 화면 한 결정 — 질문 1개만
4. 출처 항상 표시 — "@yoga_hong 인스타에서"로 신뢰 쌓기
5. 동의는 마이크로로 — S6에서 4개로 쪼개기

## 스텝별

### S0 로딩 (15초)
"잠시만요, 당신을 대신 찾아보고 있어요."
- Google/Naver/IG/사진 4개 소스 진행 체크
- 건너뛰기 링크 약하게

### S1 인물 확인
"이 세 사람 중, 어느 분이 당신인가요?"
- 최대 3 후보 카드 (신뢰도 표시)
- 기본 선택 없음 — 의식적 탭
- "제가 아닌 사람" → 모든 결과 폐기

### S2 직업·지역
"당신을 한 줄로 소개한다면?"
- 칩으로 직업 1 / 지역 1 선택
- "+ 직접 입력" 시 칩이 인풋으로 변형
- 출처 인용 italic 11px

### S3 SNS 링크
"이 링크들, 당신 것인가요?"
- 체크박스 (터치 50+)
- 신뢰도 ≥ 80% 기본 체크 on
- < 70% 경고 + 기본 off
- 직접 추가 가능

### S4 Bio 초안
"당신을 소개하는 한 줄."
- 매니페스토 톤 초안 제시
- "다른 표현으로 다시 써줘" — 3 variation 캐러셀
- 직접 쓰기 모드

### S5 프로필 사진
"어떤 사진을 프로필로 쓸까요?"
- 4~5 후보 카드 (IG 사진, Gravatar, 업로드, 이니셜)
- 저작권 마이크로 고지
- 이니셜 fallback 항상 제공

### S6 최종 동의 (발행 게이트)
"당신의 공개 페이지가 준비됐어요."
- 주소 확인 + 미리보기 썸네일
- 마이크로 동의 4개:
  - [필수] 위 초안을 제 페이지로 공개
  - [필수] imPD가 이 정보를 표시·공유하는 데 동의
  - [선택] 새 SNS 게시물 자동 반영
  - [선택] 다른 회원 추천 시 샘플로 사용
- 🎉 페이지 열기 → (confetti 1.5s) → /<slug>

## 횡단 설계
- HUD: § PROBE · 03 of 06 + 프로그레스 바
- 키보드: ←→ 이동, Enter 진행
- 건너뛰기: 24h 재개 가능 (last_step 컬럼)
- 모바일: 버튼 최소 56px, 칩 44×44
- 접근성: role/aria 전부, 2px cyan 포커스 링

## 카피 톤
- brand-dna emotional_tone 준수: 주인감·단순함·신뢰·조용한 자신감
- 영어 단독 금지, 느낌표 금지, 출처 인용 필수
- 이모지 원칙 금지 (🎉만 S6 축하 순간 예외)

## 기술
- 단일 SPA: IdentityProbesController + Turbo Frames
- Stimulus: probe_wizard_controller.js
- 상태: identity_probes.last_step / step_payloads(JSON)
- 재개 로직: last_step + 24h expires_at
