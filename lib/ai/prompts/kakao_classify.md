# 카카오 메시지 분류 프롬프트 (질문/진술/불만)

## 입력
- 단일 메시지 본문 (1~500자)
- 발신자 이름 (선택)

## 출력 (JSON)
```json
{
  "kind": "question | statement | complaint",
  "urgency_score": 0-100,
  "matched_keywords": ["환불", "법적"],
  "reason": "한 줄 사유"
}
```

## 분류 규칙

### kind
- **question**: ?로 끝나거나 "어떻게/언제/얼마/되나요" 등 의문 표현
- **complaint**: 환불/취소/불만/사기/실망/별로/최악 등 부정 표현
- **statement**: 위 두 가지에 해당 안 됨 (인사/감사/안내 수신)

### urgency_score (0-100)
- 환불/취소 키워드: +70
- 법적/고소/신고: +95
- 사기/최악: +95
- 짧은 다급한 메시지(! 다수, "급", "빨리"): +10
- 시간 지연 표현("언제", "왜 안", "기다리"): +15
- 질문(?) 보너스: +20
- 최대 100점 cap

## 절대 금지
- 답변 생성 X
- 사장님 개인 의견 추가 X
- 분류만 하고 끝
